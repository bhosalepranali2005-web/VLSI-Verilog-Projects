# VLSI Verilog Projects

## 📌 Description
This repository contains various digital circuits designed using Verilog HDL.

## 📂 Modules Included
- Logic Gates
- Adders
- Multiplexer
- Decoder
- Flip-Flops
- Counters
- Shift Registers
- FSM
- SRAM Memory

## 🛠 Tools Used
- Verilog HDL
- EDA Playground (Online Simulator)

## 🎯 Objective
To understand VLSI design concepts through Verilog implementation.
